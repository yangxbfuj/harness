# 内容空白分析

这项技能通过分析您的内容与竞争对手之间的空白来识别内容机会。找到您缺失的主题、您可以定位的关键词以及您应该创建的内容格式。

## 何时使用这项技能

- 规划内容策略和编辑日历
- 寻找快速获胜的内容机会
- 了解竞争对手在哪些方面超越您
- 识别您利基市场中服务不足的主题
- 扩展到相邻主题领域
- 优先考虑内容创建工作
- 查找竞争对手遗漏的GEO机会

## 这项技能的功能

1. **关键词空白分析**：找到竞争对手排名而您没有的关键词
2. **主题覆盖映射**：识别需要更多内容的主题领域
3. **内容格式空白**：揭示缺失的内容类型（视频、工具、指南）
4. **受众需求映射**：将空白与受众旅程阶段匹配
5. **GEO机会检测**：找到您缺失的AI可回答主题
6. **优先级评分**：按影响和努力对空白进行排名
7. **内容日历创建**：规划填补空白的内容时间表

## 如何使用

```

```


## 成功技巧

1. **专注于可操作的空白** - 并非所有空白都值得填补
2. **考虑您的资源** - 基于执行能力确定优先级
3. **质量胜过数量** - 很好地填补5个空白比糟糕地填补20个更好
4. **跟踪有效的方法** - 衡量填补空白的成功
5. **定期更新** - 空白会随着竞争对手发布而改变
6. **包括GEO机会** - 不要只优化传统搜索
