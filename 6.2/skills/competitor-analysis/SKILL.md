# 竞争对手分析

这项技能提供竞争对手SEO和GEO策略的全面分析，揭示您市场中有效的方法并识别超越竞争对手的机会。

## 何时使用这项技能

- 进入新市场或利基市场
- 基于竞争对手成功规划内容策略
- 了解竞争对手排名更高的原因
- 寻找反向链接和合作伙伴机会
- 识别竞争对手遗漏的内容空白
- 分析竞争对手的AI引用策略
- 对您的SEO性能进行基准测试

## 这项技能的功能

1. **关键词分析**：识别竞争对手排名的关键词
2. **内容审核**：分析竞争对手的内容策略和格式
3. **反向链接分析**：审查竞争对手的链接建设方法
4. **技术评估**：评估竞争对手网站健康状况
5. **GEO分析**：识别竞争对手在AI响应中的出现方式
6. **空白识别**：找到竞争对手遗漏的机会
7. **策略提取**：从竞争对手的成功中揭示可操作的见解

## 说明

当用户请求竞争对手分析时：

1. **识别竞争对手**

   如果未指定，帮助识别竞争对手：
   
   ```markdown
   ### 竞争对手识别框架
   
   **直接竞争对手**（相同产品/服务）
   - 搜索"[您的主要关键词]"并记录前5个自然搜索结果
   - 检查谁在为您的关键词做广告
   - 询问：客户将您与谁进行比较？
   
   **间接竞争对手**（不同解决方案，相同问题）
   - 搜索以问题为中心的关键词
   - 查看替代解决方案
   
   **内容竞争对手**（竞争相同关键词）
   - 可能不销售相同产品
   - 在您的目标关键词上排名
   - 包括媒体网站、博客、聚合器
   ```

## 示例

**用户**："分析为什么HubSpot在营销相关关键词上排名如此好"

**输出**：

```markdown
# 竞争分析：HubSpot

## SEO策略概览

HubSpot通过以下组合主导营销关键词：
1. **大规模内容护城河** - 10,000+博客文章
2. **免费工具作为链接诱饵** - 网站评分器、电子邮件签名生成器
3. **教育品牌** - 学院、认证、课程
4. **主题集群模型** - 开创了支柱/集群方法

## 他们成功的原因

### 内容策略

**发布量**：每月50-100篇文章
**平均字数**：2,500+字
**内容类型**：
- 深度指南（35%）
- 操作教程（25%）
- 模板和示例（20%）
- 数据/研究（10%）
- 工具和计算器（10%）

**表现最佳的内容模式**：
1. 关于广泛主题的终极指南
2. 带有电子邮件门槛的免费模板
3. 统计数据汇总文章
4. 定义文章（"什么是[term]"）

### GEO成功因素

HubSpot频繁出现在AI响应中是因为：

1. **清晰的定义**在每篇文章的开头
   > "入站营销是一种商业方法论，通过创建为客户量身定制的有价值的内容和体验来吸引客户。"

2. **可引用的统计数据**
   > "撰写博客的公司获得55%更多的网站访问者"

3. **全面的覆盖** - AI信任他们的权威

### 可链接资产

| 资产 | 反向链接 | 为什么有效 |
|------|---------|-----------|
| 网站评分器 | 45,000+ | 免费、即时价值 |
| 营销统计 | 12,000+ | 可引用的参考 |
| 博客创意生成器 | 8,500+ | 解决真实问题 |

## 可利用的弱点

1. **内容变得过时** - 许多文章超过3年
2. **通用建议** - 缺乏行业特定深度
3. **企业重点** - 服务不足的个体创业者
4. **创新缓慢** - 多年来相同的格式

## 您的机会

1. 创建他们无法涵盖的更具体、利基的内容
2. 针对他们忽略的长尾关键词
3. 在新兴领域构建交互式工具
4. 添加他们没有的原创研究
5. 专注于您利基市场中的GEO优化定义
```

## 高级分析类型

### 内容空白分析

```
向我展示[competitor]拥有而我没有的内容，按流量潜力排序
```

### 链接交集

```
查找链接到[competitor 1]和[competitor 2]但不链接到我的网站
```

### SERP功能分析

```
竞争对手赢得了哪些SERP功能？（特色摘要、PAA等）
```

### 历史跟踪

```
过去一年[competitor]的SEO策略如何演变？
```

## 成功技巧

1. **分析3-5个竞争对手**以获得全面视图
2. **包括间接竞争对手** - 他们通常有创新的方法
3. **超越排名** - 分析内容质量、用户体验
4. **研究他们的失败** - 避免他们的错误
5. **定期监控** - 竞争对手策略会演变
6. **专注于可操作的见解** - 您实际上可以实施什么？
